# Project setup

```bash
# initialize sveltekit
npm create svelte@latest my-app
cd my-app
npm install

# follow with tailwindcss setup
# https://tailwindcss.com/docs/guides/sveltekit

# initialize prisma
npm i @prisma/client
npm i prisma -D
npx prisma init

# initialize lucia-auth
npm i lucia-auth
npm i @lucia-auth/adapter-prisma
```
